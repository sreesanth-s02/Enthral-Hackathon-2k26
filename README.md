🚨 Crowdsourced Disaster Reporting Platform

A real-time disaster reporting and monitoring web application that enables citizens to report incidents and allows volunteers to track and respond efficiently using an interactive map.

📌 Problem Statement

In emergencies, there is often a delay in reporting disasters and coordinating responses.
This project solves that by providing a crowdsourced platform where users can report disasters instantly, and volunteers can monitor them in real time.

💡 Solution

Our platform allows:

📍 Users to report disasters by selecting a location on a map
📊 Volunteers/Admins to monitor incidents via a dashboard
🔥 Heatmap visualization to identify high-risk areas
⚡ Faster response and better coordination
🛠 Tech Stack
Frontend: HTML, CSS (Dark Theme), JavaScript
Backend & Database: Firebase Firestore
Maps & Visualization: Leaflet.js
Other Features: Geolocation, Real-time updates
🚀 Features

✔ Real-time disaster reporting
✔ Interactive map-based location selection
✔ Admin dashboard for monitoring
✔ Heatmap visualization of disaster zones
✔ Severity-based color indicators
✔ Dark theme UI for better UX
